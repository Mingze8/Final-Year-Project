About the Project

The topic I am carrying out now is "Cloud Based Live Delivery & Tracking System".

I have name the system as TrackNow and developed 3 different panel as following:
Client Page / Admin Page / Driver Page

The system is designed to display a Google Map for the user to track their parcel current location when their parcel is delivering.

Lastly, the system are done by HTML, CSS, JavaScript, AjaxJquery, HTML Geolocation and Google Map API.

Features of the System

Responsive Design

Automatic Map Generation

Real Time GPS Tracking

Available in any device

Responsive Design
The system is designing in a responsive way which will hugely increase user experience and readability.

Automatic Map Generation
The system will also automatic generate map with faster route plan to increase driver and admin working experience and efficiency.

Real Time GPS Tracking
The system will automatic refresh the map every 10 seconds to ensure the GPS location provided to the valuable user are accurate.
This feature allowed user to track their parcel real time and prepare for collecting their parcel.

Available in any device
The system is available in any device such as laptops, phones, tablets and desktops.
This feature allowed user of the system to access the system anytime anywhere without problem and increase their experience with the system.

User Manual

Client Page

There are two main section in the client page, user had to insert their exist tracking number in the field. The system will check with the tracking number inserted and show the current status of the parcel and map when available to the user

Notes:
The map will only show up when the parcel are currently "Out For Delivery" and the driver had logged to delivery the parcel now.

Admin Panel

There are 4 pages in the admin panel, which is the Home Page, Vehicles Page, Parcel Page and the Driver Page. Admin can access to the admin panel with a existing account and manage the data.

Driver Panel

There are 2 pages in the driver panel, which is the Home Page and the Job Page. Driver can simply change their status in the home page.

In the job page, driver can choose to deliver which parcel now by clicking the "Delivery Now" button and launch the map direction by clicking the "View Map" button.
